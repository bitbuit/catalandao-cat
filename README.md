# calendarinft-cat project

## Development

### Without Docker
```
yarn install && yarn dev
```

### With Docker
```
docker-compose -f docker-compose.dev.yml up -d
```

## Production

### Without Docker

```
yarn install && yarn dev
```

### With Docker

```
docker-compose up -d
```

### Format code

Prettify all code.

```
yarn format
```
